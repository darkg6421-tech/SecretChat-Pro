# SecretChat Pro 🔐

A premium, modern, and secure 2-person chat application built with HTML, CSS, and JavaScript, powered by Firebase Realtime Database.

## Features ✨
- **Premium UI/UX**: Polished Glassmorphism design with ambient animated backgrounds.
- **Real-time Chat**: Instant message delivery using Firebase.
- **Private Rooms**: Create or join specific rooms for 1-on-1 conversations.
- **Live Status**: See when your partner is online, offline, or typing.
- **PWA Ready**: Mobile-first design that looks like a native app.
- **Smooth Animations**: High-quality transitions and entry animations.

## Getting Started 🚀
1. Clone the repository.
2. Open `assets/js/app.js`.
3. Replace the `firebaseConfig` object with your actual Firebase project credentials.
4. Enable **Realtime Database** in your Firebase console.
5. Open `index.html` in your browser.

## Project Structure 📁
- `index.html`: Main application structure.
- `assets/css/style.css`: Premium Glassmorphism styling.
- `assets/js/app.js`: Refactored Firebase chat logic.

## Security 🛡️
Designed for privacy. No messages are stored permanently beyond the session limits (configure Firebase rules for more control).
